# Exploratory--Data--Analysis-_task5Titanic EDA Project

📌 Objective

The objective of this project is to perform Exploratory Data Analysis (EDA) on the Titanic dataset to understand patterns, trends, and relationships in the data.

🛠️ Tools Used

Python
Pandas
Matplotlib
Seaborn
📂 Dataset

Titanic Dataset (titanic.csv)
🔍 Analysis Performed

Data loading and inspection
Handling missing values
Statistical analysis
Data visualization (graphs)
📊 Visualizations

Survival count plot
Gender vs survival
Age distribution histogram
Fare vs passenger class (boxplot)
Correlation heatmap
Pairplot
💡 Key Insights

Females have higher survival rates than males.
First-class passengers survived more than others.
Higher fare passengers had better survival chances.
Age has a moderate impact on survival.
✅ Conclusion

EDA helps in understanding the dataset and identifying important patterns. Survival in the Titanic dataset is influenced by gender, class, and fare.
